    Class PurchaseFlow

    Fluxo de Compra da aplicação Sauce Demo. Esta classe contém métodos para interagir com o fluxo de compra, como adicionar itens ao carrinho, preencher o formulário de checkout, finalizar a compra e validar mensagens de confirmação.
